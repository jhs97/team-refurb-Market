# team-refurb-market

## 개요

리퍼 제품을 판매하는 쇼핑몰을 모바일앱으로 구현하기

## 구성

1. 모바일 탭 메뉴 구현
2. gnb - toggle + animation
3. 제품보기 - data fetching
4. 메인 페이지, 로그인, 장바구니, 제품페이지, 제품상세페이지 구성
